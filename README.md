Online Shopping Cart 

Objective:
To simulate adding products to cart using JSP + Servlet.

Description:
•	products.jsp displays product list
•	When user clicks “Add to Cart”, a Servlet stores product in session
•	cart.jsp displays all items added
•	Pure JSP + Servlet session handling (no payment, no DB required)

•	JSP (products.jsp, cart.jsp)
•	Servlet → Adds items to session cart
•	Session management logic
•	Dynamic cart display
